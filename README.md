A collection of battle-tested, failure-aware infrastructure playbooks designed for real-world systems, not ideal scenarios.

This repository documents step-by-step procedures for provisioning, securing, deploying, and maintaining production environments, with a strong focus on:

VPS setup & hardening
Secure access and network controls
Reverse proxy and traffic routing
DNS configuration and reliability
Deployment workflows across stacks (Node, Java, Django, etc.)
Observability, logging, and debugging
Incident handling and recovery

Each playbook is written to be:

Repeatable — usable across multiple environments
Failure-aware — includes common breakpoints and fixes
Minimal yet complete — no fluff, only what matters in production
